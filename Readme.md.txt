CF2.yaml

This file contains the cloud Formation Template for creating the Ec2-Instance,
Security Group, 
DataBase Instance, DataBase Security Group,
also installs and configures the docker and git on a EC2-Instance.

1) After creating the design template, clone git repository from github. 
   git clone <path> 
 
 --> It will ask you to enter Username And Password as repository is private. 

2) After cloning the repository. Run the following command

  Run docker-compose up -d 
  It will run all your containers and pull the images.
